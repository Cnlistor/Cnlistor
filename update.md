
## 主要修改总结

1. **命名空间统一**: 所有类、函数、变量都以 `Flex` 或 `flex` 开头
2. **命名空间使用**: 使用 `FlexTools` 命名空间封装所有代码
3. **颜色常量**: 使用 `FLEX_COLOR_` 前缀定义颜色常量
4. **枚举类**: 所有枚举类以 `Flex` 开头
5. **结构体**: 所有结构体以 `Flex` 开头
6. **方法名**: 所有方法名以 `flex` 开头或包含 `Flex`
7. **缓存变量**: 使用 `flexCache` 前缀命名缓存变量
8. **辅助方法**: 内部辅助方法以 `flex` 开头
9. **配置文件**: 配置文件名为 `flextools.conf`
10. **可执行文件**: 可执行文件名为 `flextools`

## 新特性

1. **智能缓存**: 添加了缓存机制，提高重复查询效率
2. **配置文件支持**: 支持用户配置文件自定义行为
3. **更多输出格式**: 增强了 JSON 和 CSV 输出
4. **错误处理改进**: 更完善的异常处理和错误报告
5. **模块化更好**: 每个模块都有独立的缓存和配置
6. **用户体验提升**: 添加了启动横幅和彩色输出

这个 FlexTools 版本提供了一个完整的、可扩展的系统信息收集框架，您可以根据需要继续添加更多模块和功能。
