# FlexTools

一个功能强大且灵活的 Linux 系统信息与日志收集工具。

## 特性

- **全面系统信息收集**: 系统信息、硬件配置、软件包、日志等
- **多格式输出支持**: 文本、JSON、CSV 格式输出
- **智能缓存机制**: 提高查询效率，减少系统负载
- **跨发行版支持**: 自动检测并适配 Debian/RPM 系统
- **模块化设计**: 易于扩展和维护
- **用户友好界面**: 彩色输出，清晰的帮助信息

### 快速安装
